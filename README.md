# aces-mobile
Aces is a purpose-built community operating system for Nigerians living in Sussex, UK.

Aces is a purpose-built community operating system for Nigerians living in Sussex, UK. It is not a
messaging add-on or a group chat tool — it is the infrastructure layer that connects a culturally distinct
community through trusted communication, economic opportunity, social events, and mutual support.
The core thesis
Diaspora communities are underserved by generic platforms. WhatsApp creates noise.
Facebook is fractured. Nairaland is distant. No single product exists that combines verified
local trust, cultural context, and daily utility for Nigerians in Sussex. Aces fills that gap, and
owns it.
